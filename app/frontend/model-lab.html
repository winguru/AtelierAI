Advanced endpoint overrides

ATELIER_COMFYUI_MODEL_CATALOG_URL is treated as a BASE URL (for example, http://localhost:8188). Any URI path beyond host:port is ignored, port defaults to 8188, and standard LoRA Manager endpoints are derived automatically: /api/lm/checkpoints/list and /api/lm/loras/list. You can still override with ATELIER_COMFYUI_CHECKPOINTS_URL and ATELIER_COMFYUI_LORAS_URL.
